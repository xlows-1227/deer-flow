# 生成物 Artifacts

> 模块路由：`backend/app/gateway/routers/`　|　基础路径：`/api/threads/{thread_id}/artifacts`
> 认证：需要已登录的浏览器会话（Cookie）。`POST`/`PUT`/`PATCH`/`DELETE` 等写操作还需携带 CSRF 令牌（Double Submit Cookie，前端框架自动处理）。

| 方法 | 路径 | 用途 |
| --- | --- | --- |

## `GET /api/threads/{thread_id}/artifacts/{path}`

> Get Artifact File  
<a id="get-api-threads-thread-id-artifacts-path"></a>

Retrieve an artifact file generated by the AI agent. Text and binary files can be viewed inline, while active web content is always downloaded.

**路径参数**

| 名称 | 类型 | 必填 | 说明 |
| --- | --- | :---: | --- |
| `thread_id` | string | 是 |  |
| `path` | string | 是 |  |

**查询参数**

| 名称 | 类型 | 必填 | 说明 |
| --- | --- | :---: | --- |
| `download` | boolean | 否 |  |

**响应**

- **`200`** Successful Response
- **`422`** Validation Error
  - 响应体（`application/json`）：`HTTPValidationError`，字段见 [数据模型](#数据模型)。

**请求示例（cURL）**

```bash
curl -X GET '${DEERFLOW_BASE_URL}/api/threads/{thread_id}/artifacts/{path}'
```
